# 如何在 OneNote 中创建笔记本

## 问题诊断

运行测试脚本后发现：
- ✅ OneNote 正在运行
- ✅ COM API 正常工作
- ❌ **OneNote 中没有任何笔记本**

XML 输出显示：
```xml
<?xml version="1.0"?>
<one:Notebooks xmlns:one="http://schemas.microsoft.com/office/onenote/2013/onenote"/>
```

`<one:Notebooks/>` 是空的，表示没有笔记本。

## 解决方案

### 方法 1: 在 OneNote 桌面版中创建笔记本（推荐）

#### 步骤：

1. **打开 Microsoft OneNote**
   - 从开始菜单或任务栏打开 OneNote

2. **点击"文件"**
   - 在左上角找到并点击 **文件** 菜单

3. **点击"新建"**
   - 在左侧菜单中选择 **新建**

4. **选择存储位置**
   你有两个选择：

   **选项 A: OneDrive（推荐）**
   - 优点：自动同步，多设备访问
   - 缺点：需要网络连接

   **选项 B: 此电脑**
   - 优点：本地存储，离线可用
   - 缺点：无法在其他设备访问

5. **输入笔记本名称**
   - 例如："我的工作区"、"学习笔记"、"项目计划"

6. **点击"创建笔记本"**
   - 等待 OneNote 创建并加载笔记本

7. **验证笔记本已创建**
   - 你应该能在左侧看到新笔记本
   - 笔记本名称会显示在笔记本列表中

8. **在 Obsidian 中重试**
   - 回到 Obsidian
   - 点击 "Retry Detection" 按钮
   - 现在应该能看到你的笔记本了！

### 方法 2: 使用键盘快捷键

如果 OneNote 已经打开：

1. 按 `Ctrl + N` 创建新笔记本
2. 输入笔记本名称
3. 按 Enter 确认

### 方法 3: 从模板创建

1. 打开 OneNote
2. 点击 **文件** > **新建**
3. 滚动到下方，点击 **更多模板**
4. 选择一个模板（如"学生笔记本"、"项目计划"等）
5. 点击 **创建**

## 常见问题

### Q1: 我应该将笔记本存储在哪里？

**A:** 这取决于你的需求：

| 存储位置 | 优点 | 缺点 | 适用场景 |
|---------|------|------|---------|
| **OneDrive** | 自动同步、多设备访问、云端备份 | 需要网络 | 大多数用户 |
| **此电脑** | 离线可用、隐私性好 | 单设备访问、需手动备份 | 敏感信息、无网络环境 |

### Q2: 我可以创建多少个笔记本？

**A:** 理论上没有限制。建议：
- 初学者：1-3 个笔记本
- 高级用户：5-10 个笔记本
- 不要创建太多，否则难以管理

### Q3: 笔记本和分区有什么区别？

**A:** OneNote 的层级结构：
```
笔记本 (Notebook)
  └── 分区组 (Section Group) [可选]
        └── 分区 (Section)
              └── 页面 (Page)
                    └── 子页面 (Subpage)
```

- **笔记本**: 最高层级，类似一个"文件夹"
- **分区**: 笔记本内的分类，类似"标签页"
- **页面**: 分区内的具体笔记

### Q4: 为什么我创建了笔记本但还是看不到？

**A:** 可能的原因：
1. **OneNote 还在同步** - 等待几秒钟
2. **使用了错误的 OneNote 版本** - 确保是桌面版
3. **插件需要刷新** - 点击 "Retry Detection"
4. **笔记本创建失败** - 检查 OneNote 中是否真的显示了笔记本

### Q5: 我可以导入现有的笔记本吗？

**A:** 可以：
1. 打开 OneNote
2. 点击 **文件** > **打开**
3. 浏览到 `.one` 文件位置
4. 选择文件并打开

1. 打开 OneNote
2. 查看左侧的笔记本列表
3. 应该能看到你创建的笔记本名称
